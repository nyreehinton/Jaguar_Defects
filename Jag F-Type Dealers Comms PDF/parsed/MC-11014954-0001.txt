Offboard Diagnostic Information System  Service (ODIS  Service ) Numbe r: VOS-25-16 
  
Subject:   ID.4 Airbag  Hotfix Instructions  Date: Feb. 5, 2025 
 
 
 
 
 
1.0 – Introduction 
The hotfix is for 2021 -2025 ID .4  
 
The hotfix  is for ID.4 Basic Settings test plan for DA 0015 (Airbag Module)  
During the test it will stop during the test plan and say:  
The following function could not be performed: Basic setting, Inertial sensor 
system  
End of test.  
 
!! The Hotfix must be removed after performing the diagnostic procedure . !! 
 
 
2.0 – Prerequisite  
ODIS Service latest Diagnostic Content version must be installed . 
To confirm the installed diagnostic content version i n ODIS Service, go to:  
 Info >    Versions >Diagnostic data ; didb_GFS -v.en_US:  xxxxxxxx X.XX.XX 
Important:  
We recommend installation of this hotfix on one diagnostic device ONLY. Hotfix must 
be removed immediately after perform ing the procedure.
