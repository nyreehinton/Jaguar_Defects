Acronyms:
• AV = Actual Values
• CU = Control Unit
• CUL = Control Unit Log
• DTC = Diagnostic Trouble Code
• E-Stand = Fresh-up head unit boot software (found in head unit control unit log)
• FUP = Fresh-up
• FW = Firmware
• HSVL = High Speed Video Line
• HU = Head Unit
• HW = Hardware
• ISU = Initial Start Up
• LVDS = Low Voltage Differential Signal
• NSFT = Not Safe For Transport
• MOPF = Model Facelift (Ex: 167 MOPF)
• PiH = Plug-in Hybrid
• P3 Hybrid = AMG Plug-in Hybrid with rear drive unit
• SA Code = Sonderausstattungscode (Option Code)
• SCN = Software Calibration Number
