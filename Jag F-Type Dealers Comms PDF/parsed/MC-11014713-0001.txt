Introduction  
Mack Trucks has identified that certain  MMDE model trucks may encounter charging 
issues when using a Heliox brand charger. To address this, a General Vehicle  Control Unit 
(GVCU ) software update has been developed . Please refer to the instructions in this 
document to install the update.

This bulletin applies to vehicles built within the specified production changes below.

Important Note  
Before proceeding with the repair, ensure you have the cvFlash application , all needed 
equipment,  and have received the software file from eService.

2. Apply parking brake.  
3. Place the transmission in park or neutral.  
4. Turn off the ignition.  
5. Install wheel chocks.   
 
 
 
 
 
 
6. Connect the cvFlash license flash drive to your PC.
