SERVICE:
DO NOT REPLACE any components for this issue. 
Due to a bug in the latest SW release (3.3.16,17,18,19), the function for ETA and Distance to 
Destination in the DIM is not shown but still visible in the map-view in CSD-screen.
The plan is to reintroduce this function in the next software release that is expected in year 2025 
quarter 1 in VIDA.
  
Warranty claim info:
No warranty claim is accepted for this TJ.
VEHICLE REPORT:
Yes, please submit a Vehicle Report if the service solution described in this TJ has no effect. Use 
concern area “Vehicle Report” and sub concern area “Support not needed” , use function group 
3900.
