IMPORTANT
When receiving a vehicle for service:
• Always lookup the vehicle identification number (VIN) on the Knowledge Center to ensure all required recalls are performed.
• Always connect the vehicle to the BRP diagnostic software (BUDS) to ensure all required updates are performed.
PROBLEM
The current softwares do not not allow the following features:
• Vehicle information to be displayed in the BRP GO! app.
• Heated grips activation.
The software updates will also implement various improvements for the EV modules.
SOLUTION
Perform the following software updates using BUDS3:
• VCU
• BMS
• Cluster (Multifunction Gauge)
• OBC (Only early production vehicles are affected. If this update is not listed, it means the correct software was installed at the
factory).
• INVT (Inverter)
Printed in Canada (cbg2025-001).
