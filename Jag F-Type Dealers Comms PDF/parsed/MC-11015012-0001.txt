Media Source Freezes Screen  
APPLIES TO  
Year  Model Trim  Level  VIN Range  
2024  Prologue  ALL  ALL 
SYMPTOM 
When switching to Sirius XM media source, the center display screen could freeze. The screen will remain frozen  with a 
“Loading” message displayed.  
POSSIBLE CAUSE  
The software within the radio unit may need an update. 
CORRECTIVE ACTION  
Check the software code, if needed, perform a software  update for: 
Controller Code: A11 
Controller Name:  Radio Control Module
