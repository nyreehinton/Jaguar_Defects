This Diagnostic Sheet  serves as a guide and provides additional diagnostic info for possible  issues 
related to the front door panel wiring  on Grecale ’s. 
 
MODELS:  M182 Grecale  (All MY) . 
 
TechDocs section :  
Electric System → CABLE INDEX: CA0004 → Front door panel wiring, driver side , CA0005 - Front door panel wiring, 
passenger side . Workshop Manual → 00.30 -1 → CABLES AND CONNECTORS REPAIR  00.30.005  80 
 
Customer concern : Door opening malfunction s when using the switch, but no issue when using the latch.  
 
Possible DTC  
DTCs on devices connected to the door harness; below are some DTCs for purely illustrative purposes but   
NOT exhaustive:  
DTC Code  DTC naming  
B250111  Door Internal Latch Release Switch -Circuit Short to Ground  
B250113  Door Internal Latch Release Switch -Circuit Open  
U12BD00  Lost Communication With Passenger Door Latch Module  
 
 
Issue : The door panel wiring is pinched/damaged ( during factory assembly ) in the following areas:  
• Left-hand side: between connectors D112 and wiring fork with connector D022  
• Right -hand side: between connectors D113 and wiring fork with connector D026
