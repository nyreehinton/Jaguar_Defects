 
48 V battery G1/3 with P0A7F00 - The age limit of the
hybrid/high-voltage battery module has been reached
Topic number LI54.10-P-078888
Version 1
Function group 54.10 - Battery, power supply, voltage converter
Date 2/10/25
Validity Model series 177, 118, 247 with equipment code B01
Reason for change Distribution adjusted.
Complaint
The engine warning light illuminates or the 48 V message is displayed in the instrument cluster
Cause
Fault code "P0A7F00" stored in the 48 V battery G1/3 and/or is current.
Remedy
Perform guided test for fault code "P0A7F00".
Important! To ensure a bus idle of 90 minutes is achieved, the vehicle should be locked and the key placed a mini-
mum of 30 feet from the vehicle)
See SI82.85-P-0046A
