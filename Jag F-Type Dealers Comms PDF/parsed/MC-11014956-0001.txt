Subject:  Airbag Module Software Configuration. B20130 0- No 
Band End Programming Occurs  Hotfix Instructions  Date: Feb. 12, 2025 
 
 
 
 
 
1.0 – Introduction 
The hotfix is for  multiple vehicles:  
• 2018- 2025 BW -Tiguan  
• 2018- 2023 CA/CM -Atlas  
• 2015- 2021 AU-Golf 
• 2022-2025 SD/CG -Golf 
• 2022- 2025 CL- Taos  
• 2018- 2025 BU Jetta  
 
If you have the DTC: B201300: No band end programming  occurs,  and you attempt  the test 
plan and get the error message below. This hotfix will address that.  
 
 
 
 
(continued)  
Important:  
We recommend installation of this hotfix on one diagnostic device ONLY. Hotfix must 
be removed immediately after perform ing the procedure.

2. Navigate t o Elsa2Go>Service References>Software/Hardware Updates 
3. Search for  the SWK Hotfix Zip File 
4. Click the  SWK  Hotfix Zip File    and save  it to your Windows d esktop. 
5. Close your browser .
