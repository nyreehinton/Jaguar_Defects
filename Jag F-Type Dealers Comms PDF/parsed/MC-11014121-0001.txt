DESCRIPTION:

The customer may state that when they have used ERS function in Volvo Cars App,  the vehicle 
doesn’t go up to Driving Mode:
• ERS used to start the engine
• Engine is still running after ERS
• Vehicle does not go to Driving mode when holding brake pedal and turning the Start knob
Please also view the attached video for a description/verification of the complaint.
See advise under Service.

SERVICE:
*Do not replace any parts or perform fault tracing. This will be resolved with software planned to be 
released in 2025 quarter 2.
Customer can be advised as follows:
To be able to put the vehicle into Drive mode, the driver must first shut the vehicle off (release brake 
pedal, turn Start knob), and then start the engine normally.
