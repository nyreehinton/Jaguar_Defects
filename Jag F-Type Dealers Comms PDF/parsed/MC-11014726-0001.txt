 
Fault code P06DA00 Current or Stored in engine control unit
Topic number LI07.70-N-078824
Version 1
Function group 07.70 - Gasoline injection and ignition system direct injec-
tion
Date 2/11/25
Validity 447 and 907 with engine M274
Reason for change
Complaint
Fault code P06DA00 is found current or stored in engine control unit. Check engine light may be activated.
Cause
Various causes
Remedy
Up to Model year 2018
Fault code P06DA00 can be disregarded if not activating the check engine light , unless otherwise specified in the gui-
ded test. The vehicle continues to operate as designed. Further causes should be investigated if complaint of check
engine light is received.
Model year 2019 and later
Fault code P06DA00 will activate check engine light as of model year 2019. No driveability concerns will be observed,
only illumination of the CEL.
This fault code indicates high resistance or open in the circuit to Y130 oil pump control valve.
Perform guided test for component Y130 under list of tests in ME. Diagnose location of high resistance or open in
Y130 circuit and repair as necessary.
Warranty claims submitted without supporting guided test results may be subject to debit.
