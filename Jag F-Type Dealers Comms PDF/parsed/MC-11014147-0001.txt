Please check the tire pressures, if these are OK calibrate the iTPMS system.
*A software solution is introduced from version 3.3.16, perform a software upgrade according to TJ 
35361.
*After the basic checks, initiate TPMS Store Pressure using the following method:
*Start the engine and with vehicle standing still initiate calibration of Tire Monitoring system according 
to Owners Manual.
*The car should then be driven between 50-100 km/h (31-62 mph) for 20 minutes and then the tire 
pressure should be stored as the preferred tire pressure.
