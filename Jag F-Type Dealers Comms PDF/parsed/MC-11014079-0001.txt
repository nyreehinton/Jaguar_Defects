FIRE SUPPRESSION FITTING  RETROFIT  
Retrofit Description:  
This procedure replaces a  fitting in the fire suppression system that has the potential to rub against 
other components.

Procedure  
1. Complete the Proterra approved Lockout/Tagout procedure to make the bus safe for work.  
 
2. Using a 4mm Allen  Driver, uninstall the aux heat fill plate.  
 
 
3. Using a 7/8 -Inch Combination Wrench, remove the fittings shown in the following illustration.  
  
 
4. Using a 1-1/16 Inch and a 1-Inch Combination Wrench, remove the Nozzle (029036) from the 
90-Degree Fitting ( 031514).  Retain the nozzle for later use.  
 
Remove Fitting

5. Using a 13/16- Inch and a 7/8- Inch Combination Wrench, install Straight Fitting ( 051926) onto 
Elbow fitting ( 051929).  
    
 
6. Using a 1 -1/8 Inch Combination Wrench and a 1- Inch Combination Wrench, install the Nozzle 
(029036) onto the Straight Fitting (051926).  
    
 
7. Using a 7/8- Inch Combination Wrench, install the assembled fittings back in place of the 
fittings  that were removed.
