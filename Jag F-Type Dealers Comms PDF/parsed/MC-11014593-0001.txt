 
System Inoperative on MBUX Headunit 6 Display High
(Code 549) and Augmented Reality Camera (Code U19)
Topic number LI54.71-P-078802
Version 1
Function group 54.71 - Camera and radar-assisted assistance systems
Date 2/3/25
Validity This LI is valid for all NTG6 vehicles.
Reason for change SA code added
Complaint
MBUX headunit display with red text box "System Inoperative".
ONLY  for vehicles with SA code (549+21U) or (549+U19).
Cause
Under analysis.
Remedy
For chassis 118, 167, 177, 213, 238, 243, 247, 253, and 293 use Xentry to update headunit (A26/17).
Target software number is 247 904 6600 (E905.4).
