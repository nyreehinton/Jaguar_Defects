Units : DSR -III Equipped Units  
Description : DSR -III microprocessor control boards will be replaced by DSR 3.1 
microprocessor control boards due to obsolete components on the circuit board. 
Form, fit, and function are the same as DSR -III. DSR 3.1 requires new software 
(Msk841.xx) and new parameter files  and is not backwards compatible with DSR -
III. 
Important:  Msk841.xx has a built -in safety feature to prevent it from being loaded into the 
wrong platform.  
Microprocessor Control Board Identification : 
The microprocessor control board can be identified by the Part Number of the board.  
Note: For new control boards, the r elays will be white.  
• 12V = 453003  
• 24V = 453004
